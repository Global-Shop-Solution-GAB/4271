PROCESS DOCUMENTATION AND TRAINING

Order Entry > View > Sales Order Approval Dashboard


Document Notes :
This custom project will allow the user to flag Sales order lines as approved or not approved from a dashboard and from the Sales Order Lines screen. Sales orders that are not approved will be removed from Automatic Work Order Generation and Automatic Purchasing.
Select CheckBox





Navigate to Order Entry > File > Sales Order > New/Open and go into the lines screen for an order. The Order Lines screen will have a new approval checkbox on the section where the order and line information is in the header. This will allow the user to approve a sales order line.
When an existing line is populated, it will populate this checkbox according to the saved data.
The approval flag will be saved upon saving the line.



Clicked Area





There will be a brand new custom menu item in Order Entry > View > Sales Order Approval Dashboard. Clicking on this button will summon the sales order dashboard.



Unknown Control





The Sales Order Approval dashboard has three different tabs. The first two show Open Sales Order lines in the system and the third shows Sales statistics.

The Open Order lines tabs displays information about the sales order lines. The project will automatically check the user's security group and will allow edit on the screen if the user is part of the EditSO security group.

Clicking on any item in the Order # column under the edit or View mode will populate the sales order screen from the global shop menu with the order information.

Any user is able to modify columns for Notes, User Field 4 and User Field 5 for sales order lines in the Dashboard.

In Edit mode, the user can Approve Sales Order Lines in the dashboard, and open the sales order screen by clicking on the Sales Order Line column.

In Ship mode, the user can click on the Sales Order line to print a shipping report regarding that Sales Order.



Unknown Control





The Sales Order Statistics tab has charts regarding sales order information.



Select Export CheckBox





The top right corner of the dashboard includes buttons to refresh the data and export the current data. If the user is in the security group EditSO, the user is able to create a new Sales order from this New SO button.
